# ai_agent_news
AI Coding Agentのニュースを収集するためのリポジトリ。

Claude Code Routinesによって更新される。

## ディレクトリ構成

各AIエージェントごとにディレクトリを用意している。

```
.
├── claude_code/
├── codex/
├── copilot/
└── gemini/
```

## リリースノートのフォーマット

各エージェントのディレクトリ直下に、バージョン名のファイルを作成する。

### ファイル命名規則

```
{agent_dir}/{version_name}.md
```

例:
```
claude_code/claude-code-1.0.0.md
codex/v1.0.md
```

### ファイルの内容

変更内容ごとに `##` で見出しを付け、その下に詳細を日本語で記載する。
変更内容に対応するリンクがある場合は添付すること。

```markdown
## 見出し（変更内容のカテゴリ）

詳細の説明。

[参考リンク](https://example.com)
```

### 複数バージョンのリリース

このリポジトリの最終更新後に複数バージョンがリリースされている場合は、それぞれ個別にファイルを作成する。
