## `/usage` コマンドに GitHub スタイルの貢献グラフを追加

`/usage` コマンドに GitHub スタイルの貢献グラフが追加された。ターミナルのカラーモードに適応し、カラー非対応ターミナル向けのフォールバックグリフも用意されている。

[参考リンク](https://github.com/github/copilot-cli/releases/tag/v1.0.35-3)

## レンダリングパフォーマンスの向上

タイムラインでの大きなテキスト表示のレンダリングパフォーマンスが向上した。

[参考リンク](https://github.com/github/copilot-cli/releases/tag/v1.0.35-3)

## `MULTI_TURN_AGENTS` モードでの同期タスク改善

`MULTI_TURN_AGENTS` モードで同期タスク呼び出しが完了までブロックされるようになった。従来は 60 秒後にバックグラウンドに自動昇格されていた。

[参考リンク](https://github.com/github/copilot-cli/releases/tag/v1.0.35-3)

## タブナビゲーションの改善

タブナビゲーションで Home/End キーによる最初と最後のタブへのジャンプがサポートされた。

[参考リンク](https://github.com/github/copilot-cli/releases/tag/v1.0.35-3)

## プラグインのホットリロード

プラグインがインストール後に再起動なしで即座に有効になるようになった。

[参考リンク](https://github.com/github/copilot-cli/releases/tag/v1.0.35-3)

## バグ修正

- エージェントループ内でのカスタムツール呼び出しの自己修正が改善された
- テキスト入力での絵文字およびマルチコードポイント文字のカーソル移動、削除、レンダリングが修正された
- Windows システムでのツール利用可能性検出が修正された
- ターン中のセッショントークン期限切れが自動的に処理されるようになった
- `/cwd` と `/add-dir` パスピッカーでの初回タブおよび矢印キーナビゲーションが修正された
- IDE/拡張機能の切断時に一時的な I/O エラーが赤いタイムラインエントリとして表示されなくなった
- `~/.claude/` 内のカスタムエージェントとスキルが Copilot プロジェクト設定として誤ってロードされる問題が修正された
- ログインコマンドが認証後に対話型入力を正しく復元するようになった

[参考リンク](https://github.com/github/copilot-cli/releases/tag/v1.0.35-3)
