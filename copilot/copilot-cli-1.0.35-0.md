## HTTP フックのサポート

フックが HTTP POST による JSON ペイロード送信に対応した。ローカルコマンドの実行ではなく、設定した URL にペイロードを送信できるようになった。

[参考リンク](https://github.com/github/copilot-cli/releases/tag/v1.0.35-0)

## grep / glob ツールの複数パス対応

grep および glob ツールが複数の検索パスを指定できるようになった。

[参考リンク](https://github.com/github/copilot-cli/releases/tag/v1.0.35-0)

## カスタムエージェント名の表示

カスタムエージェント名がステータスラインのフッターに表示されるようになった。`/statusline` コマンドで表示の切り替えが可能。

[参考リンク](https://github.com/github/copilot-cli/releases/tag/v1.0.35-0)

## バグ修正

- サブエージェントの思考プロセスがタイムラインに表示されなくなった
- 起動ダイアログで Escape キーを押した際に発生するレースコンディションが修正された

[参考リンク](https://github.com/github/copilot-cli/releases/tag/v1.0.35-0)
