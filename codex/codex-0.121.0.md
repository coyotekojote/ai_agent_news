## マーケットプレイスインストールの拡張

マーケットプレイスのインストールが GitHub URL、Git リポジトリ、ローカルディレクトリ、直接 JSON URL に対応した。より柔軟なプラグイン・拡張機能のインストールが可能になった。

[参考リンク](https://github.com/openai/codex/releases/tag/v0.121.0)

## TUI 履歴機能の強化

TUI に `Ctrl+R` によるリバース検索とローカルスラッシュコマンドの履歴呼び出し機能が追加された。

[参考リンク](https://github.com/openai/codex/releases/tag/v0.121.0)

## メモリモードコントロール

TUI およびアプリサーバーインターフェースの両方にメモリモードコントロールが追加された。

[参考リンク](https://github.com/openai/codex/releases/tag/v0.121.0)

## MCP／プラグインサポートの拡張

Apps ツール呼び出し、名前空間付き登録、並列実行オプションにより MCP／プラグインサポートが拡張された。

[参考リンク](https://github.com/openai/codex/releases/tag/v0.121.0)

## Realtime API の改善

出力モダリティとトランスクリプト完了イベントに対応し、Realtime API が改善された。

[参考リンク](https://github.com/openai/codex/releases/tag/v0.121.0)

## セキュア devcontainer プロファイル

bubblewrap サンドボックスをサポートするセキュアな devcontainer プロファイルが追加された。

[参考リンク](https://github.com/openai/codex/releases/tag/v0.121.0)

## バグ修正

- macOS サンドボックスでのプライベート DNS 解決処理が修正された
- Windows で `resume --last` およびスレッド操作のパスマッチングが安定化された
- Guardian のタイムアウト動作がポリシー拒否と区別されるようになり、ユーザーガイダンスが改善された
- アプリサーバーのスレッドアンロードとシンボリックリンク処理が改善された
- MCP の遅延ツール名や名前空間記述に関するエッジケースが修正された

[参考リンク](https://github.com/openai/codex/releases/tag/v0.121.0)
