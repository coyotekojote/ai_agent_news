## スラッシュコマンドポップアップの安定化

スクロール中にスラッシュコマンドポップアップの説明カラムがずれる問題が修正され、コマンドをナビゲートしても列が正しく整列されるようになった。

[参考リンク](https://github.com/openai/codex/releases/tag/rust-v0.126.0-alpha.3)

## ターミナルリサイズ時のリフロー対応

スクロールバック内容のターミナルリサイズ時リフロー処理が実装された。ターミナルのサイズが変更された際に折り返しテキストが正しく再配置されるようになり、長期間の既知の問題が解消された。

[参考リンク](https://github.com/openai/codex/releases/tag/rust-v0.126.0-alpha.3)

## エスカレートされたコマンドのネットワーク処理改善

エスカレートされたコマンド実行時にマネージドネットワークプロキシがインターセプトしないよう変更された。エスカレートされた実行はファイルシステムとネットワークの両方で Codex サンドボックス制約を完全にバイパスするようになった。

[参考リンク](https://github.com/openai/codex/releases/tag/rust-v0.126.0-alpha.3)

## `end_turn` フィールドのサポート

レスポンス完了時の `end_turn` フィールドに対応した。プロバイダーがターン遷移を明示的に示すことが可能になり、未提供の場合は従来のロジックにフォールバックする。

[参考リンク](https://github.com/openai/codex/releases/tag/rust-v0.126.0-alpha.3)

## バグ修正・改善

- Windows での app-server テストにおけるパスアサーションが安定化され、異なるテキスト形式で表現される等価な正規パスが正しく処理されるようになった
- Bazel ビルドの 30 分タイムアウトが復元され、テストシャーディングが増加された
- npm レジストリの検証が追加され、パッケージが完全に公開される前に早期のアップグレードプロンプトが表示されなくなった

[参考リンク](https://github.com/openai/codex/releases/tag/rust-v0.126.0-alpha.3)
