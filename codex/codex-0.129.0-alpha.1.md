## 自動レビューで常時許可 MCP ツールのバイパス

自動レビューモードで常時許可（always-allow）に設定された MCP ツールのレビューがバイパスされるようになった。

[参考リンク](https://github.com/openai/codex/releases/tag/rust-v0.129.0-alpha.1)

## 拡張履歴の永続化設定変更

`persist_extended_history` がデフォルトで `false` に設定された。

[参考リンク](https://github.com/openai/codex/releases/tag/rust-v0.129.0-alpha.1)

## リモートインストール済みプラグインバンドルの同期

リモートにインストールされたプラグインバンドルが同期されるようになった。

[参考リンク](https://github.com/openai/codex/releases/tag/rust-v0.129.0-alpha.1)

## 危険なプロジェクトレベル設定キーの無視

危険なプロジェクトレベルの設定キーが無視されるようになり、セキュリティが強化された。

[参考リンク](https://github.com/openai/codex/releases/tag/rust-v0.129.0-alpha.1)
