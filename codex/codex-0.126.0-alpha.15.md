## Windows コアシェル環境変数の拡充

Windows でのシェル環境ポリシー `inherit = "core"` で不足していた環境変数セットが拡充され、MCP クライアントと同等の機能的な環境変数セットが利用可能になった。

[参考リンク](https://github.com/openai/codex/pull/20089)

## フィードバックアップロードにオートレビューロールアウトを含める

`/feedback` でのログアップロード時にライブのオートレビュートランクロールアウトデータが含まれるようになった。添付データは `auto-review-rollout-<parent-thread-id>.jsonl` として保存される。

[参考リンク](https://github.com/openai/codex/pull/20064)

## リモートインストール済みプラグインキャッシュのスキルおよび MCP への適用

リモートインストール済みプラグインの状態がフェッチ・キャッシュされ、ローカルマーケットプレースエントリなしでスキルのロードが可能になった。プラグイン操作（一覧、起動、インストール、アンインストール）で非同期キャッシュ無効化と MCP リフレッシュが行われるようになった。

[参考リンク](https://github.com/openai/codex/pull/20096)

## tool_suggest のトリガー条件の改善

`tool_suggest` が明示的なプラグインインストールリクエストを優先するよう改善された。他ツールとの同時呼び出しが防止され、名前空間の再編成やスキーマ・ロギングの改善も行われた。

[参考リンク](https://github.com/openai/codex/pull/20091)
