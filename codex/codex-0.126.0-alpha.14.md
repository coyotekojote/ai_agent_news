## Windows プロセス管理のエッジケース改善

Windows でのプロセス管理におけるエッジケースが改善された。

[参考リンク](https://github.com/openai/codex/pull/19211)

## TUI からコアプロトコル依存の除去

TUI（ターミナルユーザーインターフェース）からコアプロトコルへの依存が段階的に除去された（3段階のうちステップ 1〜3）。

[参考リンク](https://github.com/openai/codex/pull/20172)

## 遅延ネットワークプロキシ拒否のハンドリング

マネージドネットワークプロキシからの遅延ネットワークアクセス拒否が適切にコマンド失敗として処理されるようになった。コマンド開始後に到着するブロックされたネットワークリクエストの報告に対応し、実行中プロセスの終了とユーザーへのエラー通知が行われるようになった。

[参考リンク](https://github.com/openai/codex/pull/19184)
