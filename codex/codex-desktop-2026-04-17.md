## バックグラウンド Computer Use

Codex デスクトップアプリに「Computer Use」機能が追加された。Codex が macOS 上のアプリを視認・クリック・タイピングにより操作できるようになった。複数のエージェントを同時に実行でき、ユーザーの作業を中断しない。ネイティブアプリのテスト、シミュレーターのフロー確認、GUI のみのバグ修正などに活用できる。まず macOS で利用可能で、EU・UK ユーザーへは今後ロールアウト予定。

[参考リンク](https://openai.com/index/codex-for-almost-everything/)

## アプリ内ブラウザ

Codex アプリにアプリ内ブラウザが搭載された。ローカルホストやサインイン不要のページを開き、レンダリングされたページに直接コメントして Codex にフィードバックを指示できる。フロントエンドやゲーム開発での反復作業を高速化する。将来的にはローカルホスト以外の Web アプリケーションへのブラウザ操作にも拡張予定。

[参考リンク](https://openai.com/index/codex-for-almost-everything/)

## メモリ機能

過去のタスクから有用なコンテキストを将来のスレッドに引き継ぐ「メモリ」機能が追加された。安定した設定、プロジェクトの慣習、繰り返しの作業パターンなどを記憶し、パーソナライズされた支援を実現する。コンテキストに応じたサジェスト機能も含まれ、Codex の開始時や復帰時に関連するフォローアップを提案する。

[参考リンク](https://openai.com/index/codex-for-almost-everything/)

## 画像生成

`gpt-image-1.5` を使用した画像の生成・修正が可能になった。スクリーンショットとコードを統合し、プロダクトコンセプトのビジュアル作成、フロントエンドモックアップのデザイン、ゲームアセットの開発などに活用できる。

[参考リンク](https://openai.com/index/codex-for-almost-everything/)

## オートメーションの強化

Codex が将来の作業を自律的にスケジュールし、長期間にわたるタスクを自動的に再開できるようになった。数日から数週間にわたるタスクにも対応する。

[参考リンク](https://openai.com/index/codex-for-almost-everything/)

## 90 以上の新規プラグイン

スキル、アプリ統合、MCP サーバーを組み合わせた 90 以上のプラグインが新たにリリースされた。Atlassian Rovo（JIRA 管理）、CircleCI、CodeRabbit、GitLab Issues、Microsoft Suite、Neon by Databricks、Remotion、Render、Superpowers などが含まれる。

[参考リンク](https://openai.com/index/codex-for-almost-everything/)

## プルリクエストワークフローの統合

GitHub プルリクエストのワークフローがアプリ内に統合された。サイドバーで PR の検査、diff 内のコメント確認、変更ファイルのレビューが可能になり、Codex にフィードバックの説明や変更の実施を依頼できる。

[参考リンク](https://openai.com/index/codex-for-almost-everything/)

## タスクサイドバーとアーティファクトビューア

タスクサイドバーにより、Codex 作業中のプラン、ソース、生成物、サマリーの追跡が容易になった。アーティファクトビューアでは、PDF、スプレッドシート、ドキュメント、プレゼンテーションなどの生成ファイルをコミットや共有前にサイドバーでプレビューできる。

[参考リンク](https://openai.com/index/codex-for-almost-everything/)
