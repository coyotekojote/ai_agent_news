## スタンドアロンインストールとデスクトップ統合

スタンドアロンインストールがより自己完結型になり、`codex app` が Windows および Intel Mac で Desktop を正しくオープン・インストールできるようになった。

[参考リンク](https://github.com/openai/codex/releases/tag/rust-v0.122.0)

## TUI の強化

`/side` コマンドによるサイドカンバセーション機能が追加された。バックグラウンドで作業が実行中でもスラッシュコマンドやシェルプロンプトによるキューイング入力が可能になった。

[参考リンク](https://github.com/openai/codex/releases/tag/rust-v0.122.0)

## プランモードの改善

プランモードがフレッシュなコンテキストで実装を開始できるようになった。プランニングスレッドを継続するかどうかの判断前にコンテキスト使用量の見積もりが表示される。

[参考リンク](https://github.com/openai/codex/releases/tag/rust-v0.122.0)

## プラグインワークフローの強化

プラグインワークフローにタブブラウジング、インライン有効化/無効化トグル、マーケットプレイス削除機能が追加された。リモート、クロスリポジトリ、ローカルのマーケットプレイスソースもサポートされた。

[参考リンク](https://github.com/openai/codex/releases/tag/rust-v0.122.0)

## ファイルシステムパーミッション

ファイルシステムパーミッションで deny-read glob ポリシー、マネージド deny-read 要件、プラットフォームサンドボックス強制、ユーザー設定やルールを無視する分離された `codex exec` 実行がサポートされた。

[参考リンク](https://github.com/openai/codex/releases/tag/rust-v0.122.0)

## ツール検出と画像生成のデフォルト有効化

ツール検出と画像生成がデフォルトで有効になった。MCP と `js_repl` の画像出力に対する画像ハンドリングとオリジナル詳細メタデータのサポートが強化された。

[参考リンク](https://github.com/openai/codex/releases/tag/rust-v0.122.0)

## セキュリティ改善

- ログアウト時にマネージド ChatGPT トークンが取り消されるようになった
- プロジェクトフックと exec ポリシーが信頼済みワークスペースを要求するようになった
- Windows サンドボックスセットアップで過度なユーザープロファイルおよび SSH ルート権限が回避された
- セキュリティ境界に関するリファレンスが `SECURITY.md` に追加された（サンドボックス、承認、ネットワーク制御）

[参考リンク](https://github.com/openai/codex/releases/tag/rust-v0.122.0)

## バグ修正

- App-server の承認、ユーザー入力プロンプト、MCP エリシテーションが他のクライアントで解決された際に TUI から消えるようになった
- リモートコントロールの起動が ChatGPT 認証の欠落を許容するようになった
- MCP 起動のキャンセルが app-server セッションで正常に機能するようになった
- 再開・フォークされた app-server スレッドでトークン使用量が即座にリプレイされ正確なコンテキスト/ステータス UI が初期化されるようになった
- サンドボックド `apply_patch` の書き込みが分割ファイルシステムポリシーで正しく動作するようになった
- ファイルウォッチャーが監視開始後に作成されたファイルを検出するようになった
- TUI の致命的なスキルリスト失敗、無効な再開ヒント、重複コンテキストエントリ、メニューループ、冗長な通知、iTerm2 でのターミナルタイトル処理が修正された

[参考リンク](https://github.com/openai/codex/releases/tag/rust-v0.122.0)

## インフラストラクチャとリファクタリング

プラグインおよびマーケットプレイスロジックが `codex-core-plugins` に、コネクタロジックが `connectors` クレートに分離された。大規模なコアモジュールの分割、設定読み込みの抽象化リファクタリング、Bazel と CI の安定化が実施された。

[参考リンク](https://github.com/openai/codex/releases/tag/rust-v0.122.0)
