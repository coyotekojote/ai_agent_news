## 動的サンドボックス拡張と worktree サポート

Linux と Windows の両方で、サンドボックスの動的拡張および Git worktree サポートが実装された。権限が制限された環境下でも、柔軟な開発ワークフローが可能になる。

[参考リンク](https://geminicli.com/docs/changelogs/latest/)

## ツールベースのトピックグルーピング（Chapters）

エージェントとの対話をツール利用や意図ごとに論理的にグループ化する「Chapters」機能が導入された。長時間セッションでも、会話の流れをより明確に把握できるようになる。

[参考リンク](https://geminicli.com/docs/changelogs/latest/)

## ブラウザエージェントの強化

ブラウザエージェントに永続的なセッション管理、動的な読み取り専用ツールの発見、サンドボックス対応の初期化機能が追加され、より堅牢で柔軟なブラウザ操作が可能になった。

[参考リンク](https://geminicli.com/docs/changelogs/latest/)

## 各種バグ修正とドキュメント更新

CLI、ブラウザエージェント、コア機能、UI 要素など、広範なコンポーネントで多数のバグ修正とドキュメント更新が行われた。

[参考リンク](https://geminicli.com/docs/changelogs/latest/)
